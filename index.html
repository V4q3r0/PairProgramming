Bienvenidos al juego del Número Mágico!!

  • Para ganar debes adivinar el número.
  • El numero debe estar entre 1-100.
  • Para cancelar debes escribir 0.
  • Existen 3 niveles y un determinado numero de intentos
  • Tienes 5 intentos para fácil.
  • Tienes 3 intentos para medio.
  • Tiens 1 intento para difícil.
